# Changelog

All notable changes to the "Simple Image Editor" extension will be documented in this file.

## [1.0.0] - 2025-06-01

### Added

- **Viewer Mode**: Pan & zoom for image viewing
- **Editor Mode**: Full editing toolbox
- **Drawing Tools**: Select, Marquee, Brush, Text, Zoom
- **Layer System**: Add, delete, duplicate, merge, reorder with 12 blend modes
- **Filters**: Gaussian Blur, Box Blur, Motion Blur with live preview
- **Clipboard**: Copy, Cut, Paste with automatic layer creation
- **Fill & Stroke**: Color fill and stroke for selections
- **Undo/Redo**: Up to 50 history steps with visual history panel
- **AI Image Generation**: OpenAI DALL-E 3 and Google Imagen support
- **Format Support**: Open PNG, JPG, JPEG, SVG, GIF, ORA; Save as PNG, JPEG, GIF
- **Rust/WASM Engine**: High-performance pixel operations
- **Secure API Key Storage**: VSCode SecretStorage integration
